HTML:

O que é HTML?

O HTML é a linguagem base para se desenvolver qualquer site. É uma linguagem baseada em marcação, onde marcamos os elementos para definir quais informações a página vai exibir. A linguagem HTML foi criada na década de 1990 e suas especificações são controladas pela W3C (World Wide Web Consortium).

CSS:

O que é CSS?

O CSS é uma forma de estilizar uma outra linguagem de programação chamada de HTML, então antes de aprender CSS recomendo fazer como eu! Domi-ne HTML antes de tudo, ele será a sua base!

O básico:

Ao desejar aprender CSS é nescesário saber que os códigos do CSS referenciam as tags do HTML, exemplo, se usarmos a forma mais
conhecida e usar css, usando a tags style podemos ver até mesmo nesse site que o css referencia o html!
